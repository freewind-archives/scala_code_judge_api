Scala Code Judger API
=====================================

This project is providing restful API to run scala code and give result:
 
1. run scala code with provided test data
2. give result (e.g. successful or not, reason)
3. security protection 

technology stack
================

1. Scala: 2.11.0
1. Play: 2.2.3
